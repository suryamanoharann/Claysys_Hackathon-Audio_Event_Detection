# Claysys Hackathon – Audio Event Detection

## Overview
A system for real-time detection of critical audio events—such as glass breaking or dog barking—to enhance security monitoring and surveillance applications.

## Features
- Detects predefined audio events in continuous streams
- Real-time processing for timely alerts
- Scalable design for integration with security systems

## Tech Stack
- Python
- Machine Learning (Audio Event Detection models)
- Libraries: Librosa, TensorFlow/PyTorch
- Deployment: Flask / Streamlit (optional)
